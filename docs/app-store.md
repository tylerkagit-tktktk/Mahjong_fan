# App Store Copy

## App Name
麻雀番埋嚟

## Subtitle
記錄對局．整理結果．快速分享

## Description
麻雀番埋嚟是一個本機優先的麻雀對局工具。你可以快速開局、逐手記錄、結束後查看總結，並以可讀格式分享結果。

### 核心功能
- 對局管理：開新局、進行中追蹤、結束後只讀摘要。
- 手數記錄：每手結果與局風進度顯示。
- 歷史列表：按時間篩選與分組查看。
- 分享摘要：玩家排名、食糊/自摸統計、對局重點。
- 本機資料：預設儲存在裝置內。

## Privacy Short Notes
- 不需要帳號。
- 不上傳對局資料到遠端伺服器（以目前版本為準）。
- 使用者可在裝置內管理與清除資料。
