# 隱私權政策（麻雀番埋嚟 / Mahjong Fan）

最後更新日期：2026-02-23

「麻雀番埋嚟（Mahjong Fan）」是一個香港四人麻雀牌局記錄與計分工具。  
本 App 以本機使用為主，重點是讓你在自己的裝置上記錄牌局與查看統計。

## 1. 我們收集哪些資料

本 App 不會主動收集可識別個人身分的資料：

- 不需要建立帳戶或登入。
- 不會收集姓名、電話、電郵、地址、位置等個人資料。
- 不會追蹤你在其他 App 或網站上的行為。

## 2. 資料儲存方式

- 牌局紀錄、玩家名稱與設定資料只會儲存在你的裝置本地。
- 本 App 不會把你的牌局內容或個人資料傳送到任何第三方伺服器。
- App 可以在離線狀態下正常使用。

## 3. 網絡與第三方服務

- 本 App 沒有內置廣告 SDK。
- 本 App 沒有內置 analytics 或 tracking SDK。
- 本 App 不會將你的遊戲內容或個人資料傳送給第三方。

## 4. 權限

目前 iOS / Android 版本不要求敏感權限（例如通訊錄、相簿、位置等）作為基本功能前提。

## 5. 分享與備份

- 當你主動使用分享功能時，內容只會交由系統分享面板處理。
- 如裝置啟用系統備份（例如 iCloud / Google Drive 裝置備份），屬於系統層級機制並由你自行控制。

## 6. 兒童與賭博聲明

- 本 App 為記錄與計分工具，不涉及真錢交易或真實賭博服務。
- 本 App 沒有向兒童作特別推廣。
